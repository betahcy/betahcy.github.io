MathJax rendering

Posted on 2020-07-06 Edited on 2021-02-25 In Hexo
Symbols count in article: 1.3k Reading time ≈ 1 mins.

https://www.latexlive.com

You’ll need to swap out the default Markdown renderer (hexo-renderer-marked) for Latex rendering without workaround escape charactors such as \, _.

Setup

Some of the guide is from Hexo Next docs

First, uninstall the default Markdown renderer (hexo-renderer-marked)

Option 1: hexo-renderer-pandoc

pandoc need to be installed in your PATH.

Option 2: hexo-renderer-markdown-it

You could use markdown-it-katex with hexo-renderer-markdown-it as in the Hexo Next docs.

However, I recommend another plugin, markdown-it-latex2img[1], which does server-side rendering of math formulae, powered by https://math.now.sh/

MathJax rendering

See MathJax quick reference for syntax.

The delimiters $, $$ follow the pandoc rule.
